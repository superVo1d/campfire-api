## Available Scripts

Install dependencies:

```
make install
```

Start server on `http://localhost:8000`:

```
make start
```

Build docker image:

```
make build
```

Freeze requirements to `requirements.txt`:

```
make freeze
```

Run app docker container:

```
make up
```

Run tests:

```
make test
```

Delete temporary files

```
make clean
```
